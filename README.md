# Design Patterns in Java

This repository contains implementations of various **Software Design Patterns** using Java.  
The goal of this repository is to understand, implement, and demonstrate commonly used design patterns in a clean and structured way.

## Purpose
- Learn and practice core design patterns
- Understand real-world use cases
- Improve object-oriented design skills
- Prepare for technical interviews

## Patterns Covered
- Factory Method Pattern
- Abstract Factory Method Pattern
### Creational Patterns
- Factory Method
- Abstract Factory
- Builder
- Singleton

### Structural Patterns
- Adapter
- Bridge
- Decorator
- Facade

### Behavioral Patterns
- Observer
- Strategy
- State
- Chain of Responsibility
- Template Method

## Structure
Each pattern is implemented in its own folder with:
- Source code
- Example usage (`Main.java`)
- Explanation (`README.md` inside each folder)

## Tech Stack
- Java
- Object-Oriented Programming (OOP)

## How to Run
1. Clone the repository
2. Navigate to any pattern folder
3. Run the `Main.java` file

## References
This repository is inspired by the book:
- Head First Design Patterns by Eric Freeman & Elisabeth Robson (O'Reilly Media)
